## 4. 毛利 ≥ 50% 的定价推论

- 不优化直接卖：定价需 ≥ **¥36–60/月**——对小程序订阅偏贵，有压力。
- **优化后的现实路径**（目标把成本压到 ¥0.3–0.4/次，即 ¥9–12/月）：
  1. TTS 用普通版音色、只在低年级/必要场景朗读（最大单项成本）；
  2. 诊断/追问等非解题轮次降档到 seed-lite 级模型（解题与判定保留 pro）；
  3. 复用上下文缓存（缓存命中输入价格是 1/5）；
  4. 同类题优先题库召回，AI 生成结果入库复用。
- 优化后定价 **¥19–25/月（或年费 ¥199 上下）即可满足毛利 ≥ 50%**，处于家长可接受区间。
